ДЕЛО «АПЕКС-Х» · ТОЛЬКО ПРОФЕССОР

Детали эксперимента — внутренний меморандум

Цель программы. Подбор биологически совместимых носителей под активный протокол Профессора. Совместимость определяется по маркерам, выявленным из образцов ДНК (отпечатки, слюна, волос).

Требования к носителю.

  • Совместимость по маркерам APX-1 / APX-4 не ниже 90%.
  • Отсутствие хронических патологий, обнаруживаемых по экспресс-анализу.
  • Предпочтительный возраст: 7–12 лет. На этом возрастном диапазоне внутренние показатели программы существенно выше — данные приведены в приложении А.
  • Низкая социальная заметность (детдом, неполная семья, район 4-Б).

Обоснование возрастного диапазона. Аналитика Ковальски за период 2014–2015 гг. указывает, что у несовершеннолетних носителей наблюдается более высокая приживаемость и стабильность результатов программы. Старший возраст даёт значительный отсев на этапе адаптации.

Процедура передачи. Оперативник доставляет совпавший образец Ковальски. Дальнейший контакт с носителем — только через его людей. Оперативнику знать имя носителя запрещено.

Зона ответственности APEX. Работаем с Ковальски. Последнее слово — за ним. APEX отвечает только за безопасность и сбор информации. Всё, что идёт дальше — не наша зона.
